## 시연 권장 동선

### 메인 (3 컷)
1. **1.jpg 업로드** → 자동 추출 → Account 보강 → 게이트 ✅ (메인 시연)
2. **6.jpg 업로드** → 일반과세자 처리 — "법인 외 개인사업자도 OK" 어필
3. **3.jpg 또는 7.jpg** → OCR 실패 → "AE가 수동 입력하세요" 메시지 → AE 입력 → 게이트 ✅ (graceful fallback 시연)

### 시드 매칭 (선택)
시드 Account 30건과 회사명/사업자번호 매칭 안 됨. 시연 매끄러움 위한 두 옵션:
- **A. 신규 거래처로 진행** — Lead 회사명을 "(주)대광건설기계"로 입력 → Convert → OCR로 Account 보강. 자연스러움. **권장**.
- **B. 시드 1건 갱신** — 시드 Account 중 1건의 회사명/사업자번호를 1.jpg와 동일하게 사전 갱신 → "기존 거래처 자동 식별" 시연. 작업 1줄.
