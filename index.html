01

한국 비즈니스 정서

입력·출력의 어휘와 형식이 한국 영업 관행에 맞아야 합니다. 사업자번호·대표자·법인명, ₩ + 일금 한글, 한국식 견적 PDF, 한국어 자연체 응답.

02

AE 의사결정 중심

AI는 추출·분류·합성만 합니다. 단가·할인·승인·납기 협상 같은 가격 결정은 AE가 직접. AI 판단은 항상 "초안"으로 표시되어 AE가 한 번 더 확인합니다.

적용 예시 재견적 자동 초안 — 라인 그대로 복제 + 협상 메모만 합성 자연어 입력바 — 데이터 인용은 시스템, 결정은 사람 사전 통보 메일 자동 초안 — 발송 전 AE 검토
03

단계별 가이드

다음에 무엇을 해야 하는지 화면이 직접 안내합니다. 단계 게이트 메시지, 5카드 콕피트, 인라인 알림 — 화면이 다음 동작을 묻습니다.

적용 예시 발굴 → 견적 게이트 (사업자번호·첨부·납기일 3-조건 점검) "납기일을 저장해주세요" 인라인 안내 장기 미수금 60일+ 상단 빨간 띠 자동 표시

3-step AI 라우터.
자연어 → Apex → 한국어 응답.

1
도구 선택 자연어 질문을 어떤 데이터로 풀지 결정합니다. Prompt: AgentRouter
2
데이터 조회 5종 Action 중 하나를 자동 선택해 Apex로 호출, 결과를 가져옵니다. Apex Invocable
3
한국어 응답 합성 조회 결과를 자연체 한국어 답변으로 변환합니다. Prompt: AgentSynthesize
키워드 라우터 + LLM 라우터 + Agentforce 응답 — 진입점은 3개, 처리는 한 곳. 어떤 경로로 들어와도 동일한 Apex 메서드를 호출합니다.
Apex · LlmRouter.cls L33 — L37 · 라우터 진입
  1. String tool = LlmRouter.choose(question); // 1단계 — 도구 선택
  2. Map<String,Object> data = invokeAction(tool, q); // 2단계 — 데이터 조회
  3. return LlmRouter.synthesize(question, data); // 3단계 — 자연어 응답
